📡 Guide Pixel Facebook
Installation étape par étape du Pixel Facebook sur FunnelAfrica. Durée : 15 minutes.
1
Créer votre Pixel Facebook
1
Ouvre business.facebook.com sur ton téléphone
2
Va dans Gestionnaire d'événements (menu ☰)
3
Clique sur + Connecter une source de données
4
Sélectionne Web → Pixel Facebook
5
Nom : FunnelAfrica CEO
URL : funnelafrica.vercel.app
💡 Ce Pixel est pour toi — pour tracker les inscriptions de nouveaux vendeurs.
6
Clique Créer le Pixel → tu reçois ton Pixel ID (15 chiffres)
⚠️ Copie ce Pixel ID — tu en as besoin pour l'étape suivante.
2
Enregistrer ton Pixel ID
Ton Pixel ID sera sauvegardé localement pour référence. Tu le colleras dans tes fichiers HTML à l'étape suivante.
3
Installer le code dans tes fichiers HTML

Colle ce code juste avant </head> dans ces fichiers :

funnelafrica-auth.htmlCompleteRegistration
funnelafrica-dashboard.htmlPageView
funnelafrica-checkout.htmlInitiateCheckout
merci.htmlPurchase
abonnement.htmlSubscribe
landing.htmlPageView

4
Événements spécifiques par page

Ajoute ces lignes dans le <script> de chaque page, après une action réussie.

PageÉvénement à ajouter
funnelafrica-auth.html
Après inscription réussie		 fbq('track', 'CompleteRegistration');
funnelafrica-checkout.html
À l'ouverture de la page		 fbq('track', 'InitiateCheckout', {value: PRIX, currency: 'XOF'});
merci.html
Après paiement confirmé		 fbq('track', 'Purchase', {value: MONTANT, currency: 'XOF'});
abonnement.html
Après choix de plan		 fbq('track', 'Subscribe', {value: PRIX_PLAN, currency: 'XOF'});
5
Vérifier que le Pixel fonctionne
1
Installe Facebook Pixel Helper sur Chrome (extension)
2
Visite funnelafrica.vercel.app
3
L'extension doit afficher un ✓ vert avec ton Pixel ID
Sur Android : vérifie dans Gestionnaire d'événements → tu dois voir "Événements reçus" avec PageView.
4
Attends 24h pour que les données apparaissent dans ton compte Ads
