🔐 Authentification

Toutes les requêtes API doivent inclure ta clé API dans le header Authorization :

Authorization: Bearer fa_xxxxxxxxxxxxxxxxxxxx # Ou via paramètre URL (moins sécurisé) GET /api/public/tunnels?api_key=fa_xxxxxxxxxxxx
Base URL : https://funnelafrica.vercel.app
📦 Format des réponses
// Succès { "tunnels": [...], "total": 5, "limit": 10, "offset": 0 } // Erreur { "error": "Clé API invalide ou expirée" }
🚀 Endpoints Tunnels
GET /api/public/tunnels Liste tes tunnels
ParamètreTypeRequisDescription
statusstringoptionnelFiltrer : "live" ou "draft"
limitintegeroptionnelNombre max de résultats (défaut: 10)
offsetintegeroptionnelPagination (défaut: 0)
curl -X GET \ "https://funnelafrica.vercel.app/api/public/tunnels?status=live" \ -H "Authorization: Bearer fa_VOTRE_CLE" // Réponse { "tunnels": [ { "id": "uuid", "name": "Ma Formation Excel", "slug": "formation-excel-pro", "status": "live", "price": 9900, "views": 245, "revenue": 98010, "orders": 10, "created_at": "2026-03-01T..." } ], "total": 1 }
📋 Endpoints Commandes
GET /api/public/orders Liste tes commandes
ParamètreTypeRequisDescription
tunnel_iduuidoptionnelFiltrer par tunnel
statusstringoptionnel"paid" | "pending" | "failed"
limitintegeroptionnelMax 100 (défaut: 20)
curl -X GET \ "https://funnelafrica.vercel.app/api/public/orders?status=paid" \ -H "Authorization: Bearer fa_VOTRE_CLE"
📊 Endpoint Statistiques
GET /api/public/stats Vue globale de tes performances
