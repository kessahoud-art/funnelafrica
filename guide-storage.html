☁️ Guide Hébergement Fichiers
Comment héberger vos PDFs, ebooks et fichiers directement dans FunnelAfrica via Supabase Storage.
1
Activer Supabase Storage
Va sur supabase.com → ton projet funnelafrica
Menu gauche → Storage
Clique New bucket
Nom du bucket : funnelafrica-files
Active Public bucket → les fichiers sont accessibles via URL
Clique Save
💡 Gratuit : Supabase offre 1 Go de stockage sur le plan gratuit. Largement suffisant pour démarrer.
2
Configurer les permissions d'accès
Dans Storage → ton bucket funnelafrica-files
Onglet Policies → New policy
Choisir For full customization
Copie ce SQL dans Supabase SQL Editor :
-- Permettre aux vendeurs connectés d'uploader leurs fichiers CREATE POLICY "vendors_upload" ON storage.objects FOR INSERT TO authenticated WITH CHECK (bucket_id = 'funnelafrica-files'); -- Permettre à tout le monde de lire les fichiers (pour les acheteurs) CREATE POLICY "public_read" ON storage.objects FOR SELECT TO public USING (bucket_id = 'funnelafrica-files'); -- Permettre aux vendeurs de supprimer leurs propres fichiers CREATE POLICY "vendors_delete" ON storage.objects FOR DELETE TO authenticated USING (bucket_id = 'funnelafrica-files' AND auth.uid()::text = (storage.foldername(name))[2]);
⚠️ Sans ces policies, l'upload échouera avec une erreur "Unauthorized".
3
Uploader un fichier dans l'éditeur
Ouvre un tunnel dans l'éditeur FunnelAfrica
Panel droit → section ⚙️ Avancé
Champ Fichier produit → clique ⬆️ Upload
Sélectionne ton fichier PDF (max 50 Mo)
Attends la barre de progression → ✓ Fichier uploadé
L'URL est automatiquement copiée dans le champ
Clique 💾 Sauvegarder
💡 Formats supportés : PDF · EPUB · ZIP · MP4 · MP3 · DOCX
4
Comment ça fonctionne automatiquement
📢 Acheteur clique "Acheter" sur ta page de vente
↓
💳 Paiement via Wave / MTN MoMo / Orange Money
↓
✅ FedaPay confirme → webhook déclenché
↓
🎉 Page merci → bouton ⬇️ "Télécharger mon produit" apparaît
↓
📚 Espace membre → bannière bleue avec bouton Télécharger
↓
📄 Fichier téléchargé depuis Supabase Storage
💡 Sécurité : Le lien est dans le page_config du tunnel — accessible uniquement depuis la page merci et l'espace membre après paiement confirmé.
5
Alternative gratuite — Google Drive
Upload ton PDF sur Google Drive
Clic droit → Obtenir le lien → Tout le monde avec le lien peut afficher
Copie le lien → colle dans l'éditeur → champ Fichier produit
💡 Quand utiliser Drive vs Supabase ?
Drive → tu démarres, fichiers volumineux, vidéos
Supabase → intégration native, fichiers < 50 Mo, plus propre
